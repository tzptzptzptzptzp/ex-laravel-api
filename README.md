## 紹介

こちらは Laravel を使用した簡易的な API です。  
エンドポイントごとに CRUD 操作に対応しています。

## 使用技術

-   [Laravel](https://laravel.com/)
-   [php](https://www.php.net/)

## 開発手順

php artisan を使用しています

```bash
php artisan
php artisan serve // サーバー起動
```
